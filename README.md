# תרגום עברי לבלאטרו / Hebrew Translation for Balatro

תרגום מלא של המשחק Balatro לעברית עם תמיכה מלאה בכתיבה מימין לשמאל (RTL) ופונט פיקסל עברי.

Full Hebrew (RTL) translation for Balatro with a custom Hebrew pixel font.

---

## דרישות / Requirements

- [Steamodded](https://github.com/Steamodded/smods/releases) >= 0.9.8
- [Lovely Injector](https://github.com/ethangreen-dev/lovely-injector/releases)

## התקנה / Installation

1. התקן Steamodded ו-Lovely Injector
2. העתק את תיקיית `HebrewBalatro` לתוך:
   - Windows: `%AppData%\Balatro\Mods\`
3. הפעל את המשחק, עבור ל-Settings → Language → עברית

---

1. Install Steamodded and Lovely Injector
2. Copy the `HebrewBalatro` folder to:
   - Windows: `%AppData%\Balatro\Mods\`
3. Launch the game, go to Settings → Language → עברית

- תרגום מלא של כל הטקסט במשחק
